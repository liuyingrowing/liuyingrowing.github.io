1.命令与配置

hexo常用命令

在根目录下
hexo s 启动本地服务
hexo d 部署到github服务仓
hexo new “博客名字用英语后面再改” 默认post格式在source/posts目录
hexo new draft “博客名字用英语后面再改” 在source/drafts目录
hexo new page 纯页面

换主题时，hexo clean清本地缓存 再执行 hexo d -g生成网页
浏览器清除缓存，刷新页面即可生效

报错

    删除 .deploy_git 文件夹，然后 hexo clean ,hexo g -d
